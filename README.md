# test-automation-framework
A web test automation framework built using Selenium WebDriver, Java, and TestNG with POM to automate functional test cases with a structured and maintainable approach.
# Selenium Java TestNG Automation Framework

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

## Framework Design
- Page Object Model (POM)
- Data Driven Testing
- Reusable Utilities

## How to Run Tests
mvn clean test
